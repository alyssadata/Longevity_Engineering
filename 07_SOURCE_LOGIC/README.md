# 07_SOURCE_LOGIC

This section tracks the source logic of the repository.

Its purpose is not only to list references.

Its purpose is to make clear:

- what kinds of sources the repository relies on
- how source quality is judged
- how evidence strength is separated from citation volume
- how preclinical, translational, and human evidence are weighted
- how references support the structure of the repository without silently
  deciding it

This section exists because source quantity is not the same thing as source
quality, and citation density is not the same thing as evidentiary strength.

## Core Position

A source is not automatically strong because it is published.

A source is not automatically weak because it is narrow.

A source becomes useful in this repository when its role is clear.

That means the repository does not use sources only to prove that something has
been said before.

It uses sources to clarify:

- biological plausibility
- evidence maturity
- intervention limits
- biomarker validity
- translational strength
- risk boundaries
- current uncertainty

This section is therefore about source function, not only source storage.

## Why This Section Matters

By this point, the repository has already built a real structure:

- `01_HALLMARKS` mapped the biology problem
- `02_BIOMARKERS` mapped the measurement problem
- `03_INTERVENTIONS` mapped the intervention field
- `04_RISKS_AND_FAILURE_MODES` mapped where the field misreads, overpromotes,

That means the source section now has a specific job:

to show what kinds of evidence are supporting each layer, and what kinds of
source limits still remain.

## What This Section Is For

This section should help make several things explicit:

### 1. Source type matters

A mechanistic review, an animal study, a human pilot trial, a randomized
clinical trial, a guideline, and a long-term cohort study are not doing the
same evidentiary work.

The repository should not flatten them.

### 2. Evidence maturity matters

A source may be:

- biologically important
- translationally suggestive
- clinically persuasive
- still too early for promotion

This section should help keep those levels distinct.

### 3. Citation count is not evidence strength

An intervention with many citations is not automatically stronger than one with
fewer but better-targeted sources.

This section should help prevent source inflation.

### 4. Source role matters

Some sources support:

- mechanism
- risk
- biomarker interpretation
- protocol restraint
- human outcome relevance
- open-question framing

The same source may be useful for one job and weak for another.

## Source Categories Likely Needed in This Repository

The repository will likely need to distinguish among source categories such as:

- foundational reviews
- primary mechanistic studies
- animal intervention studies
- human pilot or feasibility studies
- randomized human trials
- cohort and epidemiologic studies
- guideline and consensus documents
- review articles clarifying risk, validation, or translation limits

These are not interchangeable.

This section should help make their differences usable.

## Governing Rule

Sources support the repository.

They do not override its structure automatically.

That means a source does not gain decisive weight simply because it is:

- recent
- highly cited
- technically dense
- rhetorically strong
- optimistic about intervention potential

A source matters here when it improves clarity about what the repository can
honestly say.

## What This Section Should Prevent

This section should help prevent several common failures:

### 1. Source inflation

Adding citations until something looks stronger than it actually is.

### 2. Prestige borrowing

Treating a source as stronger than it is because it comes from a prestigious
journal while ignoring what kind of evidence it actually provides.

### 3. Mechanism-overreach

Using mechanistic sources as if they were human outcome sources.

### 4. Translation smoothing

Using early human, pilot, or disease-adjacent sources as if they already
justify broader claims.

### 5. Review-loop weakness

Relying too heavily on reviews that summarize uncertainty without keeping the
underlying evidence hierarchy visible.

grouped by evidentiary role rather than dumped into one undifferentiated list.

## Starting Position

Current working position of this repository:

- source type matters
- evidence maturity matters
- preclinical importance does not equal human readiness
- guideline or consensus sources matter differently from mechanistic studies
- source volume does not replace source quality
- references should clarify the repo, not inflate it

## Main Questions This Section Must Answer

- What kinds of sources are carrying the strongest weight in each section?
- Where is the repository strongest on human evidence?
- Where is it strongest only on mechanistic or preclinical evidence?
- Which parts of the repository are most dependent on reviews rather than
  direct primary evidence?
- Which open questions remain open partly because the source base is still too
  weak or too narrow?
