# 04_intervention_sources

## Intervention Sources

This file defines how sources are used to support the intervention section of
the repository.

Its purpose is not to collect all longevity-intervention literature.

Its purpose is to make clear:

- what kinds of sources support intervention classes
- what kinds of sources support mechanistic plausibility
- what kinds of sources support preclinical significance
- what kinds of sources support human translational relevance
- what kinds of sources support real functional credibility
- what kinds of sources are too weak for protocol promotion

This file exists because intervention sources do not all do the same job.

A source that shows mechanism is not the same as a source that shows human
function.
A source that shows animal lifespan extension is not the same as a source that
justifies protocol relevance.
A source that shows disease-adjacent feasibility is not the same as a source
that supports broad healthy-aging use.

This section needs those distinctions to stay explicit.

## Core Position

Intervention sources matter here not only because they provide evidence, but
because they determine what kind of status an intervention can honestly carry.

An intervention source may support:

- biological plausibility
- hallmark targeting logic
- animal efficacy
- disease-adjacent translational signal
- human feasibility
- human functional benefit
- risk clarification
- nonpromotion and restraint

Those are different evidentiary jobs.

This repository therefore does not ask only:
Is there a source for this intervention?

It asks:
What kind of authority does that source actually earn for the intervention?

## Why This File Matters

The intervention section already established several important boundaries:

- the strongest current human-grounded interventions are exercise,
  high-quality dietary patterning, and sleep and recovery treatment logic
- many other interventions remain biologically important but still below true
  protocol threshold
- exploratory importance does not equal readiness
- biomarker movement alone is not enough for promotion
- function takes precedence when biomarkers conflict with it

Those are current repository positions. :contentReference[oaicite:1]{index=1}

That means intervention sources must now be handled by evidence role, not only
by topic.

The real source question is no longer:
Can this intervention be cited?

It is:
What kind of intervention claim does the source actually justify?

## What Counts As an Intervention Source

An intervention source in this repository is a source that helps answer one of
the following questions:

- What does this intervention plausibly target?
- What kind of evidence exists for its effect?
- Is the evidence mechanistic, preclinical, translational, or human-functional?
- Is the signal disease-adjacent or broadly relevant?
- What is the main risk pattern?
- What remains too weak for promotion?
- What kind of future evidence would actually change its status?

A source is not an intervention source merely because it mentions a compound,
behavior, or therapy.

It has to clarify the intervention’s actual evidentiary position.

## Main Intervention Source Types

### 1. Mechanistic and pathway sources

These sources are useful for showing:

- how an intervention plausibly works
- which hallmarks or pathways it might affect
- why the intervention belongs in the field biologically

They are often strongest for:

- class definition
- target logic
- risk pattern identification
- early-stage plausibility

They are weaker for:
- protocol relevance
- human functional credibility
- promotion across layers

### 2. Preclinical intervention studies

These sources matter when the question is whether an intervention shows signal
in animal or model systems.

They are often strongest for:

- proof of concept
- healthspan or lifespan signal
- tissue-specific effects
- early evidence of breadth or limit

They are weaker for:
- direct human readiness
- broad protocol use
- population-fit generalization
- assuming translational success

### 3. Human feasibility and pilot studies

These sources are important because they begin the translational bridge.

They are often strongest for:

- tolerability
- feasibility
- early biomarker signal
- disease-adjacent or burden-adjacent signal
- narrowing the question of whether something is worth following

They are weaker for:
- broad efficacy claims
- stable protocol promotion
- strong general-aging conclusions
- assuming human function has already been validated

### 4. Human functional and outcome studies

These are among the most important intervention sources in the repository.

They matter when the question is whether an intervention improves things that
aging actually costs, such as:

- capacity
- resilience
- recovery
- physical function
- long-term health outcomes
- disease-free healthy aging
- real human burden reduction

These sources carry the most weight when the intervention is being considered
for stronger protocol relevance.

### 5. Risk and translation sources

These sources matter when the main question is not whether the intervention is
interesting, but what still blocks its promotion.

They are especially useful for clarifying:

- unresolved biological risk
- translation burden
- selectivity problems
- disease-context limitations
- delivery or feasibility constraints
- why an intervention remains exploratory

These sources are often some of the most important in the whole section,
because they protect the repo from quiet promotion.

## What Intervention Sources Are Good For

In this repository, intervention sources are especially useful for:

### 1. Distinguishing evidence maturity

An intervention source should help clarify whether the intervention is mainly:

- mechanistically plausible
- preclinically important
- translationally active
- human-relevant
- functionally credible
- still too weak for promotion

This is one of the most important jobs of source logic in the intervention
section.

### 2. Clarifying intervention role

A source can help show whether the intervention is strongest as:

- foundation logic
- support logic
- burden-reduction logic
- disease-adjacent logic
- exploratory frontier logic

This matters because different intervention roles require different evidence
standards.

### 3. Supporting nonpromotion honestly

Intervention sources are often most useful when they explain why something
should **not** yet move upward.

This includes sources that clarify:

- weak human evidence
- structural risk
- poor fit generalization
- biomarker-function divergence
- narrow population relevance
- high burden relative to current gain

### 4. Preventing reversal inflation

A good intervention source can help separate:

- burden reduction
- support
- improved tolerance
- narrowed risk
- partial repair
- true rejuvenation

This distinction is essential in this repository.

## What Intervention Sources Are Not Good For

Intervention sources are weaker when they are used to support claims beyond
their evidentiary scope.

For example, an intervention source is not automatically the right source for:

- proving a biomarker is individually usable
- proving protocol promotion is justified
- overruling function with mechanism
- generalizing disease-context evidence into broad healthy-aging use
- treating early human signal as if it were mature intervention status

Those are exactly the moves the repository is trying to prevent.

## How the Intervention Section Uses Sources

The intervention section should rely on sources differently at different
levels.

### At the section README level

The intervention README should rely most heavily on:

- broad intervention reviews
- translational overviews
- sources clarifying evidence maturity differences across classes

This is where the repository sets the intervention field clearly.

### At the individual-file level

Each intervention file should rely most heavily on:

- mechanistic sources for target logic
- preclinical studies for proof of concept
- human studies for feasibility, function, or translational relevance
- risk and translation sources for what still blocks promotion

This is where evidentiary layering matters most.

### At the protocol-interface level

When an intervention is close to support or protocol relevance, the source
burden becomes heavier.

The most important sources then become:

- human functional studies
- outcome studies where available
- fit-sensitive translation sources
- sources clarifying burden and risk limits
- sources clarifying why promotion is or is not earned

This is where the repository must remain strictest.

## Intervention Source Failure Modes

### Failure Mode 1 | Mechanism inflation

A mechanistic source is treated as if it already supports human relevance.

### Failure Mode 2 | Animal-to-human smoothing

A strong preclinical source is treated as if the human burden has already been
crossed.

### Failure Mode 3 | Pilot overpromotion

A small or early human source is treated as if it supports broad protocol
movement.

### Failure Mode 4 | Disease-context overgeneralization

A source with narrow disease-adjacent relevance is treated as if it supports
general healthy-aging use.

### Failure Mode 5 | Functional underweighting

A source showing real human function is treated as less important than a more
technically impressive mechanistic or biomarker source.

This repository rejects that weighting.

## What This File Should Eventually Contain

As the repository grows, this file should likely hold or point toward the
sources doing the most structural work for the intervention section.

Likely source groups include:

- intervention-class defining reviews
- major preclinical proof-of-concept studies
- human feasibility and pilot studies
- human functional or outcome studies
- translation and risk-constraint sources
- sources clarifying why specific intervention classes remain exploratory,
  support-worthy, or more protocol-relevant

The point is not to collect every intervention paper.

The point is to identify which sources actually support the intervention
ranking and maturity logic used in this repo.

## Intervention Source Questions the Repository Should Ask

Before giving an intervention source real weight, the repository should ask:

- What kind of intervention claim is this source actually supporting?
- Is this source about mechanism, preclinical signal, translation, or human
  function?
- Is it being used for a wider claim than it supports?
- Does it support intervention ranking, or only intervention interest?
- Does it support promotion, or mainly continued nonpromotion?
- Is a stronger human or risk-focused source needed for this point?

If those questions cannot be answered well, the source may be doing less real
work than it appears to do.
