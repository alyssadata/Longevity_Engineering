# 01_biology_questions

## Biology Questions

This file tracks the biological questions the repository has not yet resolved.

These are not random scientific curiosities.

They are the unresolved biology questions that still affect:

- hallmark hierarchy
- intervention ranking
- biomarker interpretation
- risk boundaries
- protocol confidence

This file exists because the biology problem is no longer undefined.

The repository has already mapped the major hallmarks and identified several
cross-hallmark patterns.

Now the question is not “what is aging?”

The better question is:

What parts of the biology remain unresolved enough to matter for the structure
of the repository?

## Core Position

The repository has already earned several biological conclusions.

It has not earned all of them.

This file therefore does **not** reopen what is already structurally settled.

It tracks the biological questions that remain unresolved **inside** the
current framework.

That includes questions about:

- what is upstream versus downstream
- what is causal versus amplifying
- what is reversible versus only manageable
- what is central across tissues versus context-specific
- what kinds of intervention logic might actually act above the hallmark layer

## Biology Questions the Repository Has Already Partly Settled

Before naming what remains open, this file should be clear about what is
already provisionally established.

Current repository-level biological conclusions include:

- aging behaves like a network of interacting failures, not one isolated cause
- sensing, regulation, and adaptive response may be as important as structural
  damage itself
- the environment ages with the cells inside it
- cancer suppression and longevity maintenance appear to be in real tension
- epigenetic reprogramming is structurally unusual because it may operate above
  individual hallmarks rather than only within one of them

These are not final truths.
They are current repository positions.

This file begins from there.

## Main Open Biology Questions

### 1. Is there a true upstream driver of aging?

This is one of the biggest biology questions in the repository.

Possibilities include:

- damage accumulation
- epigenetic information loss
- sensing and regulatory failure
- systemic-environment deterioration
- some combination that cannot be cleanly separated

The repository has already moved away from one-mechanism thinking.

But it has not yet settled whether one of these domains is more upstream than
the others in a way that should reorganize the whole framework.

### 2. Is aging primarily loss of structure or loss of regulation?

Pattern 1 suggested that decline in sensing, regulation, and adaptive response
may be at least as important as structural damage.

That is a major clue.
It is not yet a fully settled biological conclusion.

The open question is whether aging is better understood as:

- accumulated damage that eventually overwhelms regulation
- loss of regulation that then accelerates damage
- a reciprocal loop where neither can be treated as fully primary

This matters because different answers would change which interventions seem
most structurally important.

### 3. Which hallmarks are truly causal, and which are mainly amplifiers?

The hallmarks framework is useful.
It is not a causal hierarchy yet.

Some hallmarks may be:

- more upstream
- more downstream
- more context-dependent
- more amplifying than initiating

For example:

- chronic inflammation may act as a major amplifier
- dysbiosis may be partly causal and partly reflective
- cellular senescence may be both a burden and a response
- mitochondrial dysfunction may be a driver, consequence, or both depending on
  tissue and stage

This is still unresolved.

### 4. How central is the systemic environment relative to cell-intrinsic aging?

Pattern 5 established that the environment ages with the cells.

That is already an important shift in the repo.

But a stronger biology question remains open:

How much aging burden is actually maintained by the aged environment itself,
even when individual cells or pathways are improved?

This question matters because it affects whether future interventions should
target:

- cells directly
- systemic burden directly
- tissue environment first
- or some sequence between these

### 5. Is epigenetic reprogramming actually acting above the hallmarks?

The repository currently treats partial epigenetic reprogramming as
structurally unique because it may act at the level above the individual
hallmarks.

That is one of the most important biological hypotheses in the entire repo.

It is not yet settled.

The open question is whether reprogramming is truly:

- resetting a governing regulatory layer
- affecting multiple hallmarks secondarily from one higher point
- or simply appearing unusually broad because it perturbs many systems at once

That distinction matters enormously for both biology and risk.

### 6. How deep is the cancer-longevity tradeoff?

Pattern 2 already established that the cancer entanglement is not a
coincidence.

What remains open is how deep that tradeoff actually goes.

Questions include:

- Is aging partly the price of tumor suppression in a fundamental sense?
- Are the systems that preserve long function structurally inseparable from the
  systems that prevent malignant escape?
- Which interventions are merely brushing against that boundary, and which are
  built directly into it?

This question affects how far any true reversal framework can go without
becoming biologically unstable.

### 7. Which biological layers are actually reversible?

The repository currently distinguishes between:

- burden reduction
- support
- repair
- reset
- reversal

But biology has not yet cleanly told us where those lines really are.

Open questions include:

- Which hallmarks are meaningfully reversible?
- Which are only partly reversible?
- Which are more realistically manageable than reversible?
- Which changes reflect permanent loss versus recoverable state?

This matters because the word “reversal” should not be used more broadly than
the biology earns.

### 8. How much does tissue heterogeneity limit all global aging claims?

The repository already takes tissue heterogeneity seriously.

But a stronger biological question remains open:

Is a single organism-level aging model always going to be partly wrong because
organs, niches, and systems age too differently from each other?

If so, then the biology of aging may need to be understood as:

- partially unified
- partially organ-specific
- and only conditionally compressible into one whole-body story

This matters for both biomarkers and interventions.

### 9. What is the relationship between burden reduction and true rejuvenation?

Some interventions may reduce burden without actually restoring youthful
biology.

Others may improve one layer of function without resetting the underlying age
state.

The open biological question is:

What distinguishes:

- less burden
- better compensation
- more resilience
- real repair
- partial rejuvenation

The repository needs this distinction to stay sharp as it moves forward.

### 10. Are there biological thresholds rather than smooth aging decline?

The hallmarks framework often sounds continuous, but biology may not behave
that smoothly.

Possible open questions include:

- Are there tipping points where burden becomes self-reinforcing?
- Are there thresholds beyond which recovery logic changes category?
- Are there points where a hallmark becomes much more causally dominant?
- Are there windows where intervention is more biologically leverageable than
  others?

This matters for sequencing, escalation, and intervention timing.

## Biology Questions That Matter Most for the Repository

Not every biology question carries equal architectural weight.

The current highest-priority questions appear to be:

1. Is there a true upstream driver, or only a network without one center?
2. Is regulation failure more central than structural damage?
3. How deep is the cancer-longevity tradeoff?
4. Is epigenetic reprogramming actually acting above the hallmarks?
5. How much aging burden is maintained by the environment rather than by cells
   alone?
6. What does real reversibility biologically mean in this repository?

These are the questions most likely to change the whole framework if answered
more strongly.

## What Would Count As Progress on These Questions

This repository should not treat “more papers” as enough.

Progress on a biology question should mean something more specific, such as:

- stronger causal ordering
- clearer distinction between driver and amplifier
- better tissue-specific mapping
- better separation between burden reduction and true reset
- stronger evidence that one intervention logic acts above multiple hallmarks
- stronger clarity about which biological tradeoffs are structural and which
  may soften with better control

That is the kind of progress this section should look for.

## Biology Failure Modes This File Should Prevent

This file should help prevent several common failures:

### Failure Mode 1 | False closure

Speaking as if the biology is more settled than it is.

### Failure Mode 2 | Endless reopening

Reopening questions the repository has already provisionally settled enough to
use as working structure.

### Failure Mode 3 | One-cause collapse

Trying to force the biology into one master explanation before the evidence
earns it.

### Failure Mode 4 | Reversal inflation

Using “reversal” language where the biology still only supports burden
reduction, support, or partial repair.

### Failure Mode 5 | Tissue flattening

Talking about the organism as if every tissue is aging the same way at the
same rate under the same constraints.
